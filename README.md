Q1. **Write a SQL query to retrieve all columns for sales made on '2022-11-05.**
Sales on a specific date Retrieve all sales made on 5th November 2022.
```sql
SELECT 
    *
FROM
    retail_sales
WHERE
    sale_date = '2022-11-05';
```
Q2. **Write a SQL query to retrieve all transactions where the category is 'Clothing' and the quantity sold is more than 10 in the month of Nov-2022.**
High-quantity clothing sales Find Clothing category transactions with higher quantity sold in Nov 2022.
```
SELECT 
    *
FROM
    retail_sales
WHERE
    category = 'Clothing'
        AND sale_date >= '2022-11-01'
        AND sale_date < '2022-12-01'
        AND quantiy >= 2;
```
Q3. **Write a SQL query to calculate the total sales (total_sale) for each category.**
Total sales by category Calculate total revenue generated by each product category.
```
SELECT 
    category,
    SUM(total_sale) AS total_sales,
    COUNT(*) AS total_order
FROM
    retail_sales
GROUP BY category;

```
Q4.  **Write a SQL query to find the average age of customers who purchased items from the 'Beauty' category.**
Customer demographics Find the average age of customers purchasing Beauty products.
```
SELECT 
    category, ROUND(AVG(age), 2) AS avg_age
FROM
    retail_sales
WHERE
    category = 'Beauty';

```

High-value transactions Identify transactions where total_sale > 1000.

SELECT

Q6. **Write a SQL query to find the total number of transactions (transaction_id) made by each gender in each category.**
Gender-based analysis Count total transactions by gender and category.

```sql
SELECT 
    category,
    gender,
    COUNT(transactions_id) AS total_transactions
FROM
    retail_sales
GROUP BY category , gender
ORDER BY 1 ;

```
Q7.**Write a SQL query to calculate the average sale for each month. Find out best selling month in each year.** 
Time-based performance
Calculate average monthly sales
Identify the best-selling month for each year using window functions.
```sql
SELECT 
     YEAR,
     MONTH,
     avg_sales 
		FROM(
   SELECT 
        YEAR(sale_date) AS year,
		MONTH(sale_date) AS month ,
	    ROUND(AVG(total_sale),2) as avg_sales,
        RANK() OVER(PARTITION BY YEAR(sale_date) 
        order by avg(total_sale) DESC ) as rnk
      FROM retail_sales
GROUP BY  YEAR , MONTH
)t
WHERE rnk = 1
ORDER BY YEAR ;
```

Q8.** Write a SQL query to find the top 5 customers based on the highest total sales.** 
Top customers Find the Top 5 customers based on highest total sales.
```sql
SELECT 
    customer_id, SUM(total_sale) AS total_sale
FROM
    retail_sales
GROUP BY 1
ORDER BY 2 DESC
LIMIT 5;
```

Q9. **Write a SQL query to find the number of unique customers who purchased items from each category.**
Customer reach Calculate unique customers per category.
```sql
SELECT 
    category, COUNT(DISTINCT customer_id) AS cnt_unique_customer
FROM
    retail_sales
GROUP BY category;
```
